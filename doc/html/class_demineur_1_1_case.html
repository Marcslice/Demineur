Detailed Description

Représente une case du démineur. Le grille génére sa grille en créant "Lignes X Colonne" cases. Bool esTuBombe détermine si un case est une bombe ou non. Bool estOuverte détermine si la case est ouverte ou non. int nbDanger détermine combien de bombe entoure la case ouverte. Case[] casesVoisines contient les cases qui entourent cette case.
